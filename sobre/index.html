O Projeto

O BreakingTheMovies é um site de catálogo de filmes desenvolvido utilizando apenas HTML, CSS e JavaScript. Nosso objetivo é oferecer uma experiência visual envolvente e organizada, permitindo que os usuários descubram lançamentos, clássicos e produções independentes em um só lugar.

Equipe

  • Nicole Greison: HTML, CSS, Estrutura do site, Layout e conteúdo
  • Eduardo Cullmann: JavaScript, Funcionalidades interativas
  • Leonardo Gaboardi Porto Machado: Design visual, imagens e responsividade

Tecnologias Utilizadas

  • HTML5
  • CSS3
  • JavaScript (vanilla)
